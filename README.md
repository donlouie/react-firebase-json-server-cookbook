## About The Project

Simple application to test fetch api w/ json-server & firebase on react framework.

### 🏗️ Built With

-   React (create-react-app)

### 🛠️ Installation

1. Clone the repo
2. Install NPM packages
   ```sh
   npm install
   ```
3. Enter your Firebase config in `/src/firebase/config.js` file
